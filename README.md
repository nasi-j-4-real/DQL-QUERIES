# Employee Participation Information System

## Project Overview
This project is a database schema designed for tracking employees, departments, and their involvement in various projects.

## Contents
schema.sql: The database structure (tables and relationships)
queries_redo.sql: SQL queries for reporting and data analysis.

## How to Use
Run the `schema.sql` file in an SQLite environment to create the tables, then use the `queries_redo.sql` file to generate reports.
